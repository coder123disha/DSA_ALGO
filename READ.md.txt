# 🔍 Pattern Matching in C++

This project demonstrates two ways to search for a pattern within a text:

---

## 🚀 Algorithms Implemented

### ✅ 1. Brute Force
- Checks all possible positions in the text where the pattern may match
- **Time Complexity**: O(n * m)

### ✅ 2. KMP (Knuth-Morris-Pratt)
- Efficient string matching using the **LPS (Longest Prefix which is also a Suffix)** array
- Skips unnecessary comparisons
- **Time Complexity**: O(n + m)

---

## 🧪 Example Input

```cpp
Text = "ababcababcabc"
Pattern = "abc"

Brute Force Matches at indices: 2 7 10 
KMP Matches at indices: 2 7 10
